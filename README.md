# 台灣新冠疫情足跡比對 Chrome extension

本工具可以協助您比對您的Google定位紀錄以及確診者的足跡，如果您在確診者的足跡的300公尺的範圍內，會提醒您接近過高風險地區

## Development

```
npm install
npm run watch:dev  # dev is optional
```

## Release

```
npm install
npm run build
npm run build-zip
```
